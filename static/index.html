Kakao Maps SDK Load Error

The map could not be initialized. This usually happens if the current domain (e.g. http://localhost:3008) is not registered in the Kakao Developers console.

To fix this:
  1. Log in to Kakao Developers.
  2. Go to My Application > Platform > Web.
  3. Register http://localhost:3008 in the Web site domains list.
